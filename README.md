# todo-list-PERN
making use of postgress, express, react and node.js you get a todo list
made with the following youtube video: `https://www.youtube.com/watch?v=ldYcgPKEZC8`


## step-1
make a folder named todo-list-PERN.
open it in visiual studio code and open the terminal.

## step-2
- create a server directory, type: `mkdir server`
- go in to the server directory: `cd server`
- in the terminal, install init packages so it will keep track of all our packages in our application: `npm init`
- press enter until you get out of the questions of the installation.

## step-3
- Install the *express* library: __command on line__    (makes us able to create very quickly a server in node.js)
- install *postgress*
- install *cors* library: __ commqnd on line__          (makes sure that different domain application can interact with one an other)
  `npm i express pg cors`
